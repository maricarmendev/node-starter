# Inicializador de api con nodejs

Configuración básica de un proyecto con nodejs para iniciar una api con las siguientes tecnologias:

- Express
- Typescript
- Docker
- Prettier
- Husky
- Nodemon
- Mongodb
- Bcrypt
- Git

## Required Environments

| Key       | Default Value |
| --------- | ------------- |
| MONGO_URL | na            |
| PORT      | 3000          |
